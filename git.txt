首先 安装git 网址: https://git-scm.com/download  下载完成以后 window可能要选一些东西 mac 直接无脑下一步
使用git 方法:
    1. mac 安装完成以后 直接打开终端就可以使用命令
    2.window 点击右键 一般有两个选项 一个图形可视化 一个是命令行窗口 选择性进行操作
操作命令: 
    1. 首先要选择一个文件夹 可以在编辑器里打开命令窗口 也可以在直接在终端打开
    2. git init  初始化一个git 仓库
    3. git clone 'http://XXXX' 这个命令可以克隆别人的文件 或者称作别人的源码
    4. git add .  将所有文件提交到暂存区
    5. git add <name文件名> 将 指定 的文件提交到暂存区
    6. git rm --cache <name 文件名> 将提交到暂存区的文件  拉回工作区 但是它将不被git 跟踪
    7. git reset HEAD -- <name 文件名> 这个也可以将 暂存区的文件 拉回工作区 这个依然会被git 进行跟踪
    8. git commit -m '提交输入的文字 写你提交的理由'   将暂存区的内容提交
    9. git commit  如果直接回车 不输入文字 则出现选项界面 按 "i" 可以输入文字输入文字以后 按esc然后按:wq 就可以了 
    10. git reset head~ --solf  取消上一次的提交 但是他不可以取消第一次的提交 后面的提交可以 所以第一次提交尽量谨慎
    11. git log 查看提交日志
    12. git remote add "仓库名字" https://github.com/liu0517/test.git // 连接远程仓库
    13. git remote  // 查看连接仓库的名字
    14. git remote rename "原来的名字" "要修改的名字"   //修改仓库的名字
    15. git push "仓库名字" "分支名字"  // 提交远程厂库
        push 到github的时候 可能会让你输入 username 和 password 你直接输入就可以le 然后等着报错 
        因为 github 已经进制使用密码来认证了 所以填了也没用
        解决方法 :
            1. 登录github
            2. 点击用户头像 里面有一个 settings(设置)
            3. 然后左边有一个 小钥匙的图标(设置token) 点进去 进行设置就可以了 
            4. 设置以后请保存 因为你等会就看不到了
            5. 计算你保存了 这个因为网络或者其他原因 你也可能用不了
    16. ssh 鉴权 生成ssh秘钥
            1. cd 到 ssh    // cd ~/.ssh
            2. 创建生成秘钥命令 ssh-keygen -t rsa -b 4096 -C "推荐自己的邮箱"  
            3. 他会让你选择生成秘钥保存的路径 直接回车 默认的就行
            4. 后面就是 两次输入的密码 
            5. 创建完成以后 输入ls查看  会有一个.pub 的文件
            6. cat "你秘钥的名字 .pub那个" 去复制  
            7. 去你github 配置
